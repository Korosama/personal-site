11/07/2022 - Pywal and my configuration addiction

tags: Linux

✦ ✦ ✦

It's quite difficult to put into words how I'm feeling these days. I've fallen for every single tech/linux meme that i've came across :

  • Switching to parabola because of muh kernel blobs.
  • Ditching systemd.
  • Daily driving a Librebooted Thinkpad.
  • Using as much Suckless software as possible.
  • Using only cli/ncurses apps.

I feel like I've reached peak computing, everything I'd ever want to do is a keybind away. All I do is watching bash scripts running at the push of a button and do all my work while I listen to music.

This situation led me to develop a configooring addiction. My weekends are spent endlessly recompiling my software in order to make the font size of dwm a pixel smaller, some color a touch darker or a keybind a little closer to my fingers. Alas ! This endless, meaningless cycle's purpose was only to distract me from the inevitable realization that it was all about the journey and not the destination.

Pywal greatly helped to mitigate this behavior that I think a lot of people can relate to. This revolutionary tool takes an image as input and automatically generates a colorscheme. Simply type :

 wal -i YOUR_IMAGE

and let the script do the job. Even better than that, it generates files that perfectly fit in the configuration of a variety of software.

I've particularly took interest in these files : colors-wal-dwm.h and colors.Xresources.

Turns out that you can automatically input the colorscheme generated by pywal into your dwm config aswell as loading the .Xresources.

I - dwm

  • Delete these lines in your config.def.h :

    • static const char col_gray1[]  = "#222222";
static const char col_gray2[]       = "#444444";
static const char col_gray3[]       = "#bbbbbb";
static const char col_gray4[]       = "#eeeeee";
static const char col_cyan[]        = "#005577";
static const char colors[][3]      = {
    /                   fg         bg         border   */
    [SchemeNorm] = { col_gray3, col_gray1, col_gray2 },
    [SchemeSel]  = { col_gray4, col_cyan,  col_cyan  },
};

  • replace them by : 

    • #include "/home/YOUR_USER_NAME/.cache/wal/colors-wal-dwm.h"

Voila, when you generate a colorscheme and recompile dwm the colorscheme will automatically be used.

II - .Xresources

    This is pretty straightforward, just move the file wherever you want and xrdb it :

    cp $HOME/.cache/wal/colors.Xresources $HOME/.Xresources
    xrdb $HOME/.Xresources

Tip : If you use the suckless terminal, consider applying the Xresources patch in order to have your terminal colors automatically synced too.

Being aware of this, I did the natural thing and wrote a script to automate all of that : 

#!/bin/sh
[ -z "$1" ] && echo "Please provide an image as argument." && exit;
echo -e -e "Setting wallpaper ... \n"
xwallpaper --zoom $1 && cp /home/YOUR_USER_NAME/.cache/wal/colors.Xresources /home/YOUR_USER_NAME/.xresources && xrdb .xresources && echo -e "done. \n"
echo -e "Generating colorscheme ... \n"
wal -s -i $1 > /dev/null && sed -i '17d' /home/YOUR_USER_NAME/.cache/wal/colors-wal-dwm.h && echo -e "done. \n"
echo -e "Recompiling dwm ... \n"
cd /home/YOUR_USER_NAME/Repos/dwm/ && rm config.h && sudo make clean install >/dev/null && echo -e "done. \n"
echo -e "All done, please restart dwm."

IMPORTANT NOTE : If for some reason you configured your dwm in config.h and not config.def.h, this script will DELETE your config !!!

I think that it wouldn't be an overstatement to say that python-pywal saved me from the soy configoorer ideology. It taught me to stop chasing the omega perfect setup that would finally scratch that itch; simply because that dream setup does not exist. It is a fabrication of the mind. Just settle for something good & modular, get on with your life and projects and start to get some work done.

Speaking of projects, I would greatly take projects ideas that are not a multimedia home server and do not cost 1k€. I know my way around bash scripting, basic server setups, Arduino, Soldering. Figured it would be a cool way to make use of those skills and devellop them. However, hours of browsing Lainchan did not help me finding something useful to do.
